## Short answer

Aurvek uses a balance-based system rather than fixed subscription tiers. Every AI message costs a small amount deducted from your balance. Which models you can use depends on the prompt (AI assistant) you are chatting with and the permissions your account has -- there is no "free tier" with a separate model list. If a prompt creator has set up an initial balance for new users, you can start chatting immediately after signing up without adding funds.

## Notes

- **Balance, not subscriptions.** Each message deducts a cost from your balance based on the AI model and tokens used. Cheaper models cost fractions of a cent; premium models cost more.
- **Initial balance.** Some prompts offer an initial balance (up to $5) to new users who sign up through their landing page, letting you start at no cost.
- **BYOK (Bring Your Own Key).** If enabled on your account, you can use your own API keys from OpenAI, Anthropic, Google, or xAI. You pay the provider directly with no platform API cost. VIP prompts still require a small balance ($0.10) for the creator's markup.
- **Model availability.** Which models you can use depends on the prompt. Creators can force a specific model or curate a list. Available providers: OpenAI (GPT), Anthropic (Claude), Google (Gemini), xAI (Grok), and OpenRouter.
- **Feature access.** File uploads, image generation, TTS, and voice calls are account-level permissions set by administrators, not tied to payment status.
