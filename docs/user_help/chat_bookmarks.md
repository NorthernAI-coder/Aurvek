## Short answer

You can bookmark any message in a conversation to save it for quick access later. All your bookmarked messages are collected in the "My Bookmarks" view, grouped by conversation.

## Steps

1. Hover over a message (user or bot) to reveal the action icons.
2. Click the **bookmark icon** (flag icon). A brief check mark animation confirms the bookmark was saved.
3. To view all your bookmarks, click **My Bookmarks** in the sidebar. Messages appear grouped by their original conversation.
4. Click any conversation header in the bookmarks view to jump directly to that conversation.
5. To remove a bookmark, hover over the bookmarked message (it shows the bookmark icon permanently) and click it again. The bookmark is removed with a fade-out animation.

## Notes

- Bookmarked messages remain bookmarked even if you branch the conversation; however, the new branch starts with all bookmarks cleared.
- Messages can only be bookmarked after they have been saved (i.e., they have a message ID). You cannot bookmark a message that is still streaming.
- In the "My Bookmarks" view, the model selector and message input are hidden because it is a read-only view.
- Removing a bookmark while in the bookmarks view also removes the message from that view immediately.
