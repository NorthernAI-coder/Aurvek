## Short answer

Multi-AI Compare lets you send one message to up to 4 different AI models at once and see their responses side by side in a carousel. This is useful for comparing how different models handle the same question.

## Steps

1. Click the **+** button in the message input area to open the plus menu.
2. Select **Multi-AI Compare** (if available for your prompt).
3. In the modal that opens, select 2 to 4 AI models from the grouped list.
4. Optionally, check **Keep active for all following messages** to use Multi-AI mode for every message in the conversation, not just the next one.
5. Click **Apply** to activate the selection. The badge on the button shows the number of selected models.
6. Type your message and send it as usual. The response appears in a carousel where you can navigate between each model's answer using the left/right arrows.
7. To disable Multi-AI mode, open the modal again and click **Disable**.

## Notes

- You must select at least 2 models for Multi-AI Compare to activate.
- The maximum is 4 models per message.
- Each model's response is charged independently to your account balance.
- Multi-AI responses are stored as a single message and are included when exporting to PDF.
- You can copy, bookmark, or listen to individual model responses from the carousel using the action icons.
- The Multi-AI Compare option may not be visible if the prompt configuration does not support it.
