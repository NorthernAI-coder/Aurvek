## Short answer

To use WhatsApp with Aurvek, you need to add and verify your phone number in your account settings. The phone number you register must match the one you use to send WhatsApp messages.

## Steps

1. Go to **Settings** (or **Edit Profile**) from the main menu.
2. Find the **Phone Number** field under Account Information.
3. Enter your phone number. The country code is selected automatically based on your location, but you can change it manually.
4. Click **Send Verification Code**. An SMS with a code will be sent to that number.
5. Enter the verification code in the field that appears.
6. Click **Verify Code**. If the code is correct, your phone number is confirmed.
7. Save your profile to keep the changes.

## Notes

- Each phone number can only be linked to one account. If the number is already in use, you will need to use a different one.
- If the platform requires phone verification for WhatsApp access, you will not be able to send messages until verification is complete.
- When you try to assign a conversation to WhatsApp without a phone number, a popup will ask you to add one in Settings first.
- The phone number is stored in E.164 international format (e.g., +1234567890).
