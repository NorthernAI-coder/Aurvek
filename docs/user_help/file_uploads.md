## Short answer

You can attach images and PDF files to your messages. Use the plus (+) menu to select files, or paste an image directly from your clipboard. Attached files appear as previews before you send the message.

## Steps

1. Open a conversation that uses a vision-capable model.
2. To attach files, do one of the following:
   - Click the **+** button, then select **Attach files**. A file picker will open.
   - Paste an image from your clipboard directly into the chat (Ctrl+V or Cmd+V).
3. Select one or more image files or PDF documents.
4. A preview of the attached file appears above the message input area. For images, a thumbnail is shown. For PDFs, a badge with the file name is displayed.
5. Type your message (optional) and send. The AI will process the attached files along with your text.

## Notes

- Supported file types: images (JPEG, PNG, GIF, WebP, etc.) and PDF documents.
- PDF files must be under 25 MB.
- The attach files option only appears if the current model supports vision. If you do not see it in the plus menu, the selected model does not support file attachments.
- You can attach multiple files at once.
- Pasting images from the clipboard only works for image content, not PDF files.
