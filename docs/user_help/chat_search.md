## Short answer

You can search across all your conversations using the search bar in the sidebar. Results show matching messages with highlighted snippets, grouped by conversation, and clicking a result jumps directly to that message.

## Steps

1. Type your query in the search input at the top of the sidebar (minimum 3 characters).
2. Press **Enter** to search.
3. Results appear in the sidebar, replacing the normal conversation list. Each result shows the conversation name, message type (user/bot), date, and a snippet with matching terms highlighted.
4. Click any result to open that conversation and scroll directly to the matching message, which will be highlighted.
5. If there are more results, click **Load more** at the bottom to load the next batch.
6. Click the **X** button in the search input or clear the text to exit search mode and return to the normal sidebar view.

## Notes

- Search is powered by full-text search (FTS5) and supports quoted phrases for exact matches (e.g., `"visa interview"`).
- Results are ranked by relevance and load in batches of 30.
- The highlighted message in the chat area can be dismissed by clicking the small "x" button on the highlight.
- Search only returns messages from your own conversations.
