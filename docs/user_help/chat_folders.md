## Short answer

You can create folders in the chat sidebar to group conversations. Drag and drop any conversation onto a folder, or use the chat menu to move it. Folders have custom names and colors, and you can expand them to browse their contents.

## Steps

1. Click the **+** button in the Folders section of the sidebar to create a new folder.
2. Enter a folder name and pick a color, then save.
3. To move a conversation into a folder, drag it from the chat list and drop it onto the folder. Alternatively, click the three-dot menu on any conversation and select **Move to folder**.
4. Click a folder to expand it and see the conversations inside. Click again to collapse it.
5. To edit a folder's name or color, click the three-dot menu on the folder and select **Edit**.
6. To remove a conversation from a folder, open the chat's three-dot menu and choose **Remove from folder**, or drag it out.
7. To delete a folder, use the folder's three-dot menu and select **Delete**. All conversations inside will return to the main chat list.

## Notes

- Only one folder can be expanded at a time; expanding a folder collapses any other open folder.
- Conversations inside folders show the same context menu as regular chats (rename, download PDF/MP3, delete, etc.).
- If you start a new conversation while a folder is selected (expanded), the new chat is created inside that folder.
- Folders load their conversations in small batches. If a folder has many chats, a "Load more" button appears at the bottom.
