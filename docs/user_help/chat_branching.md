## Short answer

You can fork (branch) a conversation from any message to create a new, independent conversation that includes all messages up to that point. This lets you explore a different direction without losing your original thread.

## Steps

1. Hover over any message (user or bot) in the chat window.
2. Click the **branch icon** (a code-branch icon) that appears among the message action icons.
3. A confirmation dialog will ask if you want to create a new conversation branching from that message.
4. Click **Confirm**. A new conversation is created with all messages up to and including the selected one copied into it.
5. The new conversation opens automatically and appears in your sidebar. A notification shows how many messages were copied.

## Notes

- The branched conversation keeps the same AI model and prompt settings as the original.
- Bookmarks are not carried over to the branched conversation; they reset to unbookmarked.
- The branch icon is not shown when viewing the "My Bookmarks" screen.
- If the original conversation is inside a folder, the branch is created in the same folder.
